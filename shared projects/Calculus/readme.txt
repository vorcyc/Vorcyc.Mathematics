1.	Differentiation (导数)
•	Derivative: 计算函数的导数。
•	PartialDerivative: 计算多变量函数的偏导数。
2.	Integration (积分)
•	Integral: 计算函数的不定积分。
•	DefiniteIntegral: 计算函数的定积分。
•	NumericalIntegration: 使用数值方法（如梯形法、辛普森法）计算积分。
3.	Limits (极限)
•	Limit: 计算函数的极限。
4.	Series (级数)
•	TaylorSeries: 计算函数的泰勒级数展开。
•	FourierSeries: 计算函数的傅里叶级数展开。
5.	Numerical Methods (数值方法)
•	NewtonRaphson: 使用牛顿-拉夫逊法求解方程。
•	RungeKutta: 使用龙格-库塔法求解微分方程。
